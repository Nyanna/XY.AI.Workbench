# Eclipse Plugin - Copy Path

Implementiere eine Aktion, in einem Eclipse Plugin, für das Kontextmenü, die den absoluten Dateipfad der ausgewählten Datei oder Dateien mit "`" umschlossen in die Zwischenablage kopiert. Das Ganze ist analog dem schon vorhandenen "Copy Qualified Name".

## Kontext

* Eclipse Plugin Pfad: `/home/user/xyan/xy.ai.workbench`
* Java Root-Paket: `/home/user/xyan/xy.ai.workbench/src/xy/ai/workbench`


## Result
`/home/user/xyan/XY.Docs/any/test.md`

`/home/user/xyan/XY.Docs/any/drill.md`
`/home/user/xyan/XY.Docs/any/sand.md`
`/home/user/xyan/XY.Docs/any/test.md`
